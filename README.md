# RuRevivals Marketplace

Торговая площадка для карточек RuRevivals бота.

## Быстрый старт

1. Открой `/market` в боте
2. Выбери карточку
3. Установи цену
4. Продавай другим игрокам

## Технологии

- HTML/CSS/JavaScript
- Telegram Web Apps
- Статический хостинг

## Деплой

Развернуто на:

- **Cloudflare Pages**: https://rurevivals-market.pages.dev
